Margin & Padding

Margin and padding can be declared in one class, each separated by a dash. The first part should be m or p (margin or padding), the second part should be the number (05, 10, 15, 20, 30), and the last part should be the direction (x, y, xy, top, right, bottom, left).

  • .margin-none
  • .margin-auto
  • .p20x-p10y
  • .p05top-p20right-p15bottom-p10left
  • .p30xy

Buttons & Input Elements

Buttons and other input elements should take up 100% of the screen width (with allowances for margin) unless otherwise specified. Buttons are exactly 64px high.

.input-outline should be used for codes; .input-ghost should be used for all forms that have more than one input.

Typically, .input-search-ghost goes in the header, but there may be exceptions.

<header class="third bg-charcoal">
    <h1 class="text-center full-x">Header with search</h1>
    <div class="absolute bottom-0 left-0 full-x input-search-ghost">
        <input type="search" placeholder=".input-search-ghost"/>
    </div>
</header>
<div class="bg-shamrock text-center two-thirds p15xy">
    <input class="input-ghost full-x m15bottom" placeholder=".input-ghost"/>
    <input class="input-outline full-x m15bottom" placeholder=".input-outline"/>
    <button class="bg-charcoal uppercase m15bottom">.uppercase</button>
</div>
<button class="fixed bottom-0 bg-white-20p semibold">.fixed .bottom-0 .semibold</button>

            <label>List item</label>
            <input type="checkbox"/><span></span>

Progress Bars

Progress bars use the <progress> element with a nested .progress element for unsupported browsers. The <span> element width may need to be adjusted for unsupported browsers based on its data-value attribute. .bg-color changes the progress background and .txt-color changes the progress value.

<progress class="bg-charcoal txt-shamrock" value="50" max="100">
    <div class="progress bg-charcoal txt-shamrock"><span data-value="50"></span></div>
</progress>

Bar height can be change by adding .height-10 or .height-15, or by specifying a height to both <progress> and .progress in the CSS.

<progress class="bg-slate txt-campus height-10" value="25" max="100">
    <div class="progress bg-slate txt-campus height-10"><span data-value="25"></span></div>
</progress>

.white-round is a special bar with a transparent white background and solid white value.

<progress class="white-round height-10" value="75" max="100">
    <div class="progress white-round height-10"><span data-value="75"></span></div>
</progress>
